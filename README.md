SSLyze
======

Fast and full-featured SSL scanner.

Usage
-----

### Prerequisites: 
	Python 2.6 or 2.7 and OpenSSL 0.9.8+.

### Sample command line:
	$ python sslyze.py --regular www.isecpartners.com:443 www.google.com

User manual available at https://github.com/nabla-c0d3/sslyze/wiki
